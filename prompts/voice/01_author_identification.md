# Author Identification Analysis

## Purpose
Identify claimed authorial sections and analyze distinctive features of each claimed author's voice.

## Module
Voice Variation

## Version
1.0 (2025-01-29)

## Prompt Text

```
Analyze the following passage from the Book of Mormon, attributed to [AUTHOR NAME].

PASSAGE:
[INSERT PASSAGE]

TASK: Identify stylistic features that characterize this author's voice.

Analyze the following dimensions:

1. SENTENCE STRUCTURE
   - Average sentence length
   - Syntactic complexity
   - Use of subordinate clauses
   - Characteristic sentence patterns

2. VOCABULARY
   - Distinctive word choices
   - Vocabulary richness/diversity
   - Specialized terminology
   - Repeated phrases or formulations

3. RHETORICAL PATTERNS
   - Use of rhetorical questions
   - Direct address to reader
   - Exhortation patterns
   - Argument structures

4. THEMATIC CONCERNS
   - Topics emphasized
   - Theological focus areas
   - Narrative interests

5. EMOTIONAL TONE
   - Prevailing emotional register
   - Variation in intensity
   - Personal vs. impersonal voice

OUTPUT FORMAT:

## Stylistic Profile: [Author Name]

### Summary
[2-3 sentence summary of distinctive voice characteristics]

### Detailed Features

#### Sentence Structure
[Findings with specific examples]

#### Vocabulary
[Findings with specific examples]

#### Rhetorical Patterns
[Findings with specific examples]

#### Thematic Concerns
[Findings with specific examples]

#### Emotional Tone
[Findings with specific examples]

### Distinctive Markers
[List of features that most distinctively characterize this author]

### Comparison Notes
[How these features compare to other claimed authors, if known from prior analysis]

### Hypothesis Implications

**If these features suggest distinctive authorship:**
- Hypothesis A interpretation: [How ancient-origin proponents would explain]
- Hypothesis B interpretation: [How 19th-century-origin proponents would explain]

**If these features suggest stylistic uniformity:**
- Hypothesis A interpretation: [How ancient-origin proponents would explain]
- Hypothesis B interpretation: [How 19th-century-origin proponents would explain]

### Confidence Assessment
[High/Moderate/Low/Indeterminate] with explanation

### Limitations
[What this analysis cannot determine]
```

## Expected Output Format
Structured analysis following the template above.

## Notes
- Run this prompt for each major claimed author: Nephi (1 and 2), Jacob, Enos, Jarom, Omni (multiple), Words of Mormon, Mosiah (abridged), Alma (abridged), Helaman (abridged), 3 Nephi (abridged), 4 Nephi, Mormon, Moroni, Ether (abridged)
- Distinguish between claimed original authors and Mormon's abridgment of their records
- Consider sample size limitations for authors with short sections
